# Just Helper

Just Helper - это набор утилит и вспомогательных функций, разработанный специально для Gamesense. Проект объединяет в себе востребованные инструменты для расширения базового функционала клиента и упрощения работы пользователя.

## Основные возможности

* Библиотека функций: комплекс полезных инструментов для решения различных задач внутри клиента.
* Встроенная система обновлений: возможность поддерживать скрипт в актуальном состоянии прямо из меню игры без необходимости ручного скачивания файлов.

## Установка и запуск

1. Первоначальная загрузка скрипта осуществляется строго через 'loader
2. lua'.
3. После успешной загрузки выберите и запустите скрипт в соответствующем разделе меню Gamesense.

## Система обновлений

Управление версиями происходит непосредственно через интерфейс скрипта:

1. В меню скрипта найдите и нажмите кнопку "Check for updates".
2. Скрипт отправит запрос на сервер для проверки актуальной версии.
3. Если обновление будет найдено, скрипт автоматически скачает необходимые файлы в локальную директорию.
4. По завершении скачивания произойдет автоматическая перезагрузка Lua. Все новые изменения вступят в силу моментально, перезапуск клиента не требуется.

## Требования

* Библиотеки: PUI
* Интернет-соединение для связи с сервером обновлений.
